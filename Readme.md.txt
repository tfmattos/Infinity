# Gerenciador de Projetos

Este é um MVP para uma plataforma de gerenciamento de projetos com cadastro de projetos, features e tasks (backlog).

## Estrutura do Projeto

- **Projeto:** Entidade principal com informações básicas e resultados financeiros.
- **Feature:** Módulo ou funcionalidade dentro de um projeto.
- **Task (Backlog):** Tarefas associadas a uma feature.
